NP-Completeness

MIN-VERTEX-COVER

Description

Given undirected graph $G(V,E)$, find the minimum subset vertices that touch all edges.

Decision version (vertex cover):

Given $G(V,E)$ and integer $k$, does $G$ contain a vertex cover of size at most $k$?

MAX-INDEPENDENT-SET

Description

Given undirected graph $G(V,E)​$, find the largest subset of vertices, such that there is no edge between them.

Decision Version (Independent set):

Given $G(V,E)$ and an integer $l$, does $G$ contains an independent set of size at least $l$?

Use MIN-VERTEX-COVER to prove that MAX-INDEPEDENT-SET is NP-completeness

$Proof$:

  1. Independent-set $\in$ NP : Given certificate $V’$, we can check whether
    • $V’\subseteq V\rightarrow \Theta(\left|V’\right|)$
    • $\left|V’ \right|\geq l\rightarrow \Theta(1)$
    • $V’$ is an independent set$\rightarrow \Theta(E)$
    • These procedures can be processed in polynomial time.
