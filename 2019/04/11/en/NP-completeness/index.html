MIN-VERTEX-COVER

Description

Given undirected graph $G(V,E)$, find the minimum subset vertices that touch all edges.

Decision version (vertex cover):

Given $G(V,E)$ and integer $k$, does $G$ contain a vertex cover of size at most $k$?
